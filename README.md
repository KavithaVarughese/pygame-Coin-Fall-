# pygame-Coin-Fall-

Made using pygame.
Works on Windows with Python 2.x

# Requirements

1. pygame
2. sys
3. random
4. math
5. ctypes

# Objective of the game
1. Use your arrow keys to collect as many coins from the coin fall in 30 seconds. 
2. Blue coins are worth thrice the normal coins! 
3. Avoid the bombs!
